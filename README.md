# Project Template

Starter template for front-end projects using:
* Jade
* Stylus
* CoffeeScript (with Browserify)
* Gulp

## Prerequisites

* NodeJS / NPM
* Gulp

```
npm install -g gulp
```

## Development instructions

### Install Dependencies

```
npm install
bower install
```

### Run Dev Environment

```
gulp
```

<!-- ### Test

```
gulp test
``` -->

### Prepare Assets for Deployment

```
gulp deploy
```
